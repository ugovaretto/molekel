Radiation Spectra

Molekel can display infra-red and Raman activity spectra if the information is available in the loaded file.

To display the radiation spectrum dialog:

  1. select the molecule
  2. select the Analysis->Radiation Spectrum menu item

http://bioinformatics.org/molekel/uploads/ReferenceGuide/spectrum_dlg.png

The radiation spectrum window allows the user to select the diagram type (infra-red or Raman) and display style as well as to save the graph to PostScript or PDF formats.

Moving the mouse pointer over the graph displays frequency and intensity information at the pointer position.
