  • Gaussian 09 to 03 converter shell script - provided by Lorenz Blum. In order for this script to work you need to provide the GFINPUT, GFOLDPRINT and POP= keywords to the Gaussian calculation.

Source code: tar.gz .zip

NOTE:

The Mac OS executables require X11 due to a dependency in one of the libraries we use. The X11 environment is available on the Mac OS installation DVDs; specific information on X11 is available on the Apple website. To check if X11 is already installed on your system open the Finder->Utility folder and verify that an X11 icon is present.
